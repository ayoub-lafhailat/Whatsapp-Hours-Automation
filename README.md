# WhatsApp Hours Registration Automation Tool

## Overview

This project is a Python-based automation tool developed for DC Schoonmaakdiensten to automate the processing of employee work hours shared through WhatsApp group messages.

The tool extracts unstructured time registration information from WhatsApp conversations and converts it into structured Excel administration data.

The goal of the project was to reduce manual administrative work and improve the efficiency of work hour processing.

---

# Features

- Automated WhatsApp message extraction
- Parsing of employee start and end times
- Conversion of unstructured text into structured data
- Excel export functionality
- Automated administration workflow
- Message processing and filtering

---

# Technologies Used

- Python
- Selenium
- BeautifulSoup
- Excel automation
- XPath

---

# Project Goal

The project was created to automate repetitive administrative tasks related to employee hour registration.

Instead of manually processing WhatsApp messages, the system automatically:

1. Reads WhatsApp group messages
2. Extracts relevant work hour information
3. Structures the extracted data
4. Exports the results into Excel format

This significantly reduced manual processing time and improved consistency.

---

# How It Works

The automation workflow consists of:

- Opening WhatsApp Web using Selenium
- Navigating through selected group conversations
- Extracting message content and timestamps
- Detecting work hour patterns
- Structuring the extracted information
- Exporting the processed data into Excel spreadsheets

---

# Challenges

Some important technical challenges included:

- Handling inconsistent message formats
